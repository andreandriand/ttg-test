- Untuk menjalankan program nomor 1, 3, dan 4 cukup membuka file index.html pada browser.
    Khusus program nomor 3 dan 4, outputnya dapat dilihat pada console browser melalui inspect element.
    
- Untuk menjalankan program nomor 2, pastikan sudah menginstall node.js dan menginstall package yang diperlukan dengan cara menjalankan perintah `npm install` pada terminal di direktori nomor 2. 
    Pastikan sudah membuat database mysql sesuai dengan konfigurasi yang ada pada file api.js.
    Setelah itu jalankan perintah `node api.js` pada terminal di direktori nomor 2.
    Lalu gunakan postman untuk melakukan request ke url guna mencoba API.
